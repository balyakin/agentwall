# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our standards

- Be respectful and constructive.
- Assume good intent.
- Focus feedback on technical content.
- Do not harass, demean, or discriminate.

## Enforcement

Report issues to maintainers via security@agentwall.dev.

Maintainers may remove comments, close discussions, or ban contributors for repeated violations.
